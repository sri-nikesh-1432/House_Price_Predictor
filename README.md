# House_Price_Predictor
Machine Learning project that predicts house prices using real-world housing dataset from Kaggle.

## Features Used
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
- Google Colab

## ML Concepts Used
- Regression
- Train-Test Split
- Model Evaluation
- Feature Importance
- Prediction System

## Dataset
Housing Prices Dataset from Kaggle.
